# Mastodon


## Реєстрація :raising_hand:

Перед тим як влетіти в вечірку федіверсу вам потрібно створити акаунт, для цього дотримуйтесь простих кроків нижче.

- Перейдіть на сторінку [реєстрації](https://soc.ua-fediland.de/auth/sign_up) та ознайомтеся з усіма правилами, якщо ви згодні з ними, клікніть прийняти.
- Далі вам потрібно заповнити поля: ім'я користувача (за яким вас можна буде знайти), вашу пошту (на яку прийде лист підтвердження), пароль і кілька слів, чому ви хочете приєднатися, або описати себе (це дає змогу адміністрації розуміти, з ким ми маємо справу, і захищатися від простих ботів).
    
  ![Форма реєстрації](/image/mastodon/reg.webp)

- Перейдіть за посиланням у листі та пройдіть капчу.

  ![Лист](/image/mastodon/mail.webp)

Вітаємо, ви створили акаунт, ваш феді id буде такого вигляду @username@soc.ua-fediland.de :tada:



## Профіль :stars:

Настав час заповнити ваш профіль і показати індивідуальність!

- Першим кроком надисніть на три полоски біля нікнейму або переходимо [по лінку](https://soc.ua-fediland.de/settings/profile).

  ![Відкриття меню редагування профілю](/image/mastodon/editprof.webp)

- Заповнюєте поля по смаку (не забувайте додавати аватарку).

  ![Редагування профілю](/image/mastodon/editprof2.webp)

- Натискаєте кнопку зберегти.



## Пошук :mag:

Пошук у мастодоні працює доволі просто, для пошуку користувачів, які вже відомі серверу, достатньо ввести їхній нікнейм, наприклад бот [UA](https://soc.ua-fediland.de/@ua)

  ![Пошук бота UA](/image/mastodon/searchua.webp)

Але якщо користувач новий або сервер його не знає, потрібно ввести повний id наприклад @koch@pixelfed.in.ua це нікнейм + домен(pixelfed.in.ua), на якому розташований сервер.

  ![Пошук бота UA](/image/mastodon/searchfedilid.webp)



## Постинг :pencil:

Мастодон це платформа вільного постингу, пост підтримує такі формати контенту:

  - Текст
  - Медіа(також працює Ctrl+V)
  - Опитування(але існує обмеження або медіа або опитування)

  ![Пост](/image/mastodon/post.webp)

Ачивка перший пост :trophy:



## Додатки :iphone:

У мастодон є офіційний мобільний додаток для [Android](https://play.google.com/store/apps/details?id=org.joinmastodon.android) та [iOS](https://apps.apple.com/us/app/mastodon-for-iphone/id1571998974), доступні також сторонні додатки на всі можливі пристрої, повний список можна знайти [тут](https://joinmastodon.org/apps)

Після встановлення потрібно ввести в поле сервер/інстанс: `soc.ua-fediland.de`



## Міграція :airplane:

Мігрувати з одного сервера на інший доволі просто.

- У налаштуваннях старого акаунта подамо запит на експорт усіх даних.

  ![Експорт/імпорт](/image/mastodon/migrate.webp)

- Тепер імпортуємо в новому акаунті все, що потрібно, вибираючи з випадаючого меню.

  ![Імпорт](/image/mastodon/import.webp)
